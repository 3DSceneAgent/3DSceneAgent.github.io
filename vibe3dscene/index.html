Vibe3DScene

Create Your Own 3D Scene with Words.

GitHub Project Web Demo

Vibe3DScene turns natural language into structured 3D production. It supports headless deployment for scalable backend execution and web-native scene generation for fast interactive iteration. Scene control, retrieval, generation, and rendering are unified under one agentic runtime.

Use Case 1

Headless Backend + Web Builder

Deploy Vibe3DScene as a backend service and generate complete 3D scenes directly from the web interface.

Use Case 2

Blender Built-in Chat Generation

Create and refine scenes from inside Blender through integrated chat, while tool orchestration remains automatic in the background.

Use Case 3

Blender + Cursor

Use Blender with Cursor-style coding workflows for rapid prompt-to-code-to-scene loops across roles.
